## Usage

- Build

    ```console
    docker-compose build
    ```

- Run `main.py`

    ```console
    make run
    ```

    or with arguments

    ```console
    make run args="config.data.loader.batch_size=16"
    ```

    or with detached option

    ```console
    make drun
    ```
